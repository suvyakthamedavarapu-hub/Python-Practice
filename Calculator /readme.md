Simple Calculator :
It is a basic application built using python.
Concepts applied:
-> Conditional Statement : if,elif,else
-> Standard Input/Output Statements
Process:
It is a simple calculator used to perform the basic operations. Such as:
* Addition
* Subtraction
* Multiplication
* Division
Steps Involved:
1) The user enters 2 numbers
2) An operator to perform the operation
3) If addition the system returns the sum of the two numbers
4) If subtraction the system returns the difference of two numbers
5) If multiplication the system returns the product of two numbers
6) If division the system returns the Quotient of two numbers
7) If the operator entered is wrong or invalid it returns a statement telling the user that invalid operator is entered.
